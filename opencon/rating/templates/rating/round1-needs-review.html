Filter Applications

This tool allows you to filter Round 1 applications so that you can rate ones that match your interests and experience. You can filter by issue area, academic field, or nationality â€” or any combination thereof. To use the filter, expand the menus below and select the options you would like to filter by. Click "Filter" and a list of applications will appear. Click an application to rate it. Once you submit your rating, you will be returned to this page.

Note that the more specific you get with the filter, the fewer applications that will match. If the filter doesn't return any results, try checking more boxes, or only filtering by one type of criteria. If you would prefer to be automatically assigned applications to rate, select the "R1 Rate" tab above.
